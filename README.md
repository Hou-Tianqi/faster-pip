# faster-pip
## 让你的python环境管理更轻松

这只是一个为了让你省去打开CMD和敲命令的时间，不要指望它和uv一样快,这俩根本不是一个东西

这款工具是面向刚刚入门，想学习python的第三方库但却被cmd界面弄得晕头转向的人

下载最新的之后用python解释器运行即可，不需要第三方库

这款工具目前仍处于早期测试版本，如有BUG欢迎提一个lssues
